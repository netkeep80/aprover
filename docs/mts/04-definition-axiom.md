## Проекция 4: Аксиома определения `:` как онтологический акт различения формы

### Онтологическое различение `:` и `=`

| Оператор | Онтологический статус | Природа | Направленность |
|----------|----------------------|---------|----------------|
| `:` | **Акт различения формы** | Процесс установления соответствия между знаком и паттерном | Ориентирован: `(s : F) != (F : s)` |
| `=` | **Факт структурной тождественности** | Констатация неразличимости двух связей по форме | Симметричен: `s = F ⇔ F = s` |

**Ключевое различие:**  
Знак `s` в записи `s : F` не «ссылается» на связь — он **задаёт паттерн различения формы**. Акт `s : F` есть запрос ко всем связям, удовлетворяющим форме `F`. Это не внешнее описание, а внутренний онтологический акт в асети связей.

### Аксиома определения как порождающая связь

```
(s : F) -> (s = F)
```

Здесь `->` — не логическая импликация, а **онтологический акт порождения**: из акта различения формы возникает факт структурной тождественности. Это не вывод в метаязыке, а связь между двумя онтологическими моментами одной целостности:

1. **Момент различения** (`s : F`): знак `s` устанавливает паттерн поиска в асети
2. **Момент тождества** (`s = F`): найденная связь структурно неразличима от формы запроса

Аксиома определения фиксирует **синхронное возникновение** паттерна и его резолюции в асети. Нет цикличности, потому что нет «ссылки»: есть только запрос по форме и его удовлетворение в онтологическом поле связей.

### Почему `:` всегда влечёт `=`, но не наоборот

**Направленность `:` против симметрии `=`:**
```
(s : F) != (F : s)   // определение ориентировано
но
(s = F) ⇔ (F = s)    // тождественность симметрична
```
Если бы `:` и `=` были тождественны, возникло бы противоречие: ориентированный акт стал бы симметричным фактом.

**`=` может возникать без `:`:**

1. **Через вывод из других аксиом:**
   ```
   ♂∞ = (♂∞ -> ∞)       // из аксиомы ♂
   ∞♀ = (∞ -> ∞♀)       // из аксиомы ♀
   ----------------------------------------
   (♂∞ -> ∞♀) = ((♂∞ -> ∞) -> (∞ -> ∞♀))  // через конгруэнтность =
   ```

2. **Через унификацию компонентов:**
   ```
   ((a -> b) = (c -> d)) -> { a = c, b = d }  // аксиома декомпозиции
   ```

3. **Через инверсию:**
   ```
   (a = b) -> (!a = !b)  // конгруэнтность инверсии
   ```

Акт определения `:` — **достаточное условие** для возникновения `=`, но не необходимое. Тождественность может порождаться и другими онтологическими актами (вывод, унификация, инверсия).

### Дуальность `=` и структура связи

Символ `=` выбран вместо `≡` именно из-за его **графической дуальности**, отражающей онтологическую двойственность связи:

| Вертикаль (становление) | Горизонталь (бытие) |
|------------------------|---------------------|
| `->` порождает новую целостность из начала и конца | `=` констатирует неразличимость двух целостностей |
| Акт связывания: из `b` и `e` возникает `b -> e` | Факт тождества: `x = y` фиксирует структурную устойчивость |
| Динамика конструирования | Статика формы |

Эта дуальность не метафора — она **онтологически фундаментальна**: бытие есть различение форм в актах связывания. Связь `->` создаёт новую целостность, а тождественность `=` фиксирует её структурную устойчивость. Две горизонтальные черты `=` визуально выражают эту двойственность: две неразличимые связи, лежащие «рядом» в онтологическом поле.

### Итог

Аксиома определения не отождествляет `:` и `=`, а устанавливает **порождающую связь** между ними: акт различения формы → факт структурной тождественности. Это сохраняет онтологическую чистоту МТС и устраняет разрыв «знак ↔ связь»: знак работает как запрос по форме, а не как внешняя ссылка. Различение и тождественность — два аспекта одного онтологического поля, где всё есть связи.
