## Проекция 2: Знак как запрос по форме

### 1. Первичный акт: различение без объектов

Субъект фиксирует только последовательность различий:
```
состояние A → состояние B → различие d₁
d₁ → состояние C → различие d₂
d₂ → состояние D → различие d₃
...
```

Больше субъект ничего не различает. Нет «объектов», нет «свойств», нет «внешнего мира» — есть только акты различения. МТС делает предположение: по ту сторону субъекта существует асеть связей, и каждое различение `dₖ` соответствует некоторой связи в этой асети. Но субъект никогда не имеет прямого доступа к связям — он работает только со своими различиями.

### 2. Знак как паттерн различения формы

Классическая семиотика утверждает: знак → объект (референция). Это создаёт непреодолимый разрыв: знак существует в сознании, объект — в мире.

МТС радикально переосмысливает природу знака:

> **Знак не ссылается на связь — знак есть оператор различения формы в асете связей.**

Запись `∞ : ∞ ⟼ ∞` читается не как «символ ∞ обозначает связь ∞», а как:

> «Пусть знак `∞` задаёт паттерн поиска: найти все связи, удовлетворяющие форме `x = x ⟼ x`».

Это не ссылка, а **запрос по форме** — критерий отбора из «океана связей». Нас не волнует, сколько таких связей существует «по ту сторону» — важно, что мы можем их идентифицировать по форме.

### 3. Формализация аксиомы определения

Аксиома определения устанавливает порождающую связь между актом различения формы и фактом структурной тождественности:

```
(s : F) ⟼ (s = F)
```

Где:
- `s` — знак как паттерн различения
- `F` — форма связи (конструкция из операторов ⟼, ♂, ♀)
- `:` — конструктор акта различения формы (ориентированный: `(s : F) ≢ (F : s)`)
- `=` — констатация структурной тождественности (симметричная: `s = F ⇔ F = s`)
- `⟼` — онтологический акт порождения: из акта различения возникает факт тождественности

**Ключевое различие `:` и `=`:**
| Оператор | Онтологический статус | Природа | Направленность |
|----------|----------------------|---------|----------------|
| `:` | Акт различения формы | Процесс установления соответствия | Ориентирован |
| `=` | Структурная тождественность | Факт неразличимости по форме | Симметрична |

### 4. Почему это не цикличность, а синхронность

Классическое возражение: «`∞ : ∞ ⟼ ∞` — циклическое определение: символ определяется через самого себя».

Это возражение основано на ошибочном предположении: знак должен сначала существовать как имя, а затем ссылаться на уже существующий объект.

В МТС действует иной принцип:

> **Паттерн формы и его резолюция возникают синхронно в едином акте онтологического различения.**

Процесс резолюции запроса `∞`:

```
Шаг 1: Знак "∞" вводится как запрос по форме (акт : )
        ∞ : (∞ ⟼ ∞)

Шаг 2: Запрос разрешается в асете связей (акт ⟼ )
        ∃!x (x = x ⟼ x)  →  x есть результат резолюции

Шаг 3: Факт тождественности (акт = )
        ∞ = результат_резолюции
```

Нет цикла «имя → объект → имя», потому что:
- Знак `∞` не «ссылается» на связь — он задаёт критерий отбора
- Связь не «определяется» знаком — она удовлетворяет форме запроса
- Резолюция происходит в асете, где связи существуют независимо от знаков

Это синхронное возникновение: паттерн ⇄ форма в едином акте различения.

### 5. Устранение разрыва «знак ↔ связь»

Разрыв в классической семиотике возникает из двойственности:
- Знак — в субъективном поле (сознание)
- Объект — в объективном поле (мир)

МТС устраняет этот разрыв радикально:

> **Знак и связь существуют в одном онтологическом поле — поле различений.**

Как это работает:

```
Субъект фиксирует различие d₁ → вводит знак s₁ как паттерн формы
Субъект фиксирует различие d₂ → вводит знак s₂ как паттерн формы
...
Знак s₁ сам является различением → следовательно, знак есть связь
```

Каждый знак — это уже связь, определённая своим паттерном формы. Формальная нотация МТС не описывает связи извне — она сама есть последовательность связей, различающих формы других связей.

Разрыва нет, потому что нет разделения на «субъективное» и «объективное» — есть только онтологически единое поле различений, где знаки и связи суть два аспекта одного и того же: акта различения формы.

### 6. Пример: аксиома смысла

Запись `∞ : ∞ ⟼ ∞`:

1. **Акт различения (`:`):**  
   «Введём знак `∞` как паттерн поиска всех связей формы `x = x ⟼ x`»

2. **Порождение тождественности (`⟼`):**  
   Из акта различения возникает связь между знаком и формой

3. **Факт тождественности (`=`):**  
   Знак `∞` структурно тождественен любой связи, удовлетворяющей форме `x = x ⟼ x`

Символ `∞` слева от `:` — не «имя», а **потенциальная возможность различения**.  
Форма `∞ ⟼ ∞` справа — **конкретная структура этого различения**.  
Связка `:` — **онтологический акт самоопределения**, который создаёт устойчивость различения через самозамыкание.

Это не циклическое определение, а **генеративный акт**: возможность различения получает устойчивую форму через самозамыкание. ∞ — не «бесконечная цепочка», а неразложимая целостность, условие возможности самого различения начала и конца.

### 7. Онтологическая чистота

Подход «знак как запрос по форме» обеспечивает онтологическую чистоту МТС:

✅ Нет внешних объектов — всё есть связи  
✅ Нет скрытых параметров — знак явно задаёт форму различения  
✅ Нет разрыва «знак ↔ связь» — оба суть различения в едином поле  
✅ Нет цикличности — есть синхронное возникновение паттерна и резолюции  
✅ Самодостаточность — язык описания есть часть описываемой онтологии  

Знак в МТС — это не мост между субъективным и объективным. Знак — это инструмент различения форм в самом «океане связей». Различение становится смыслом лишь в контексте сети отношений, и знак есть паттерн этого различения.
